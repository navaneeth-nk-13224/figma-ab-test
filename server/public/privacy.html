Privacy Policy

Last updated: March 19, 2026

AB Test is a Figma plugin that lets designers create shareable voting links to compare two designs. This policy explains what data is collected and how it is handled.

What Data We Collect

  • Design Screenshots — When you create an AB test, the two selected frames are exported as PNG images and uploaded to our server.
  • Test Metadata — The test title and label names you provide.
  • Voter IP Hash — When someone votes, a one-way SHA-256 hash of their IP address is stored to prevent duplicate votes. The raw IP address is never stored.
  • Vote Choice — Which option (A or B) a voter selected.

How Data Is Used

Data is used solely to operate the voting feature — displaying designs to voters, recording votes, and showing results to the test creator. No data is sold, shared with third parties, or used for advertising.

Where Data Is Stored

All data is stored on Zoho Catalyst (Datastore and Stratus object storage), hosted in Zoho's data centers.

Data Retention

When you close a vote from the Figma plugin, all associated data is permanently deleted — including the test record, all votes, and both design images. No backups are retained.

No Account Required

Voters do not need to create an account or log in. No cookies are used. A localStorage flag is set in the voter's browser to remember their vote locally.
